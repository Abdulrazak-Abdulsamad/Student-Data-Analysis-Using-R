# Student-Data-Analysis

This repository contains an R-based data analysis project that explores student academic performance and programming language preferences using a real-world student dataset. The project demonstrates the complete data analysis workflow, from data cleaning and preprocessing to statistical analysis, visualization, and interpretation of results.

## Project Objectives

* Analyze the relationship between students' level of course understanding and their CGPA.
* Examine programming language preferences across different genders.
* Apply statistical methods to generate meaningful insights from educational data.

## Features

* Data cleaning and preprocessing using **tidyverse** and **janitor**
* Removal of duplicate records and handling of inconsistent data
* Exploratory Data Analysis (EDA)
* Boxplots and bar charts for CGPA analysis
* Stacked bar chart for programming language preferences by gender
* One-Way ANOVA and Tukey HSD post-hoc analysis
* Summary statistics and automated interpretation of findings
* Export of publication-quality visualizations

## Technologies Used

* **R**
* **tidyverse**
* **ggplot2**
* **janitor**

## Learning Outcomes

This assignment demonstrates proficiency in:

* Data wrangling and preprocessing
* Exploratory Data Analysis (EDA)
* Statistical hypothesis testing
* Data visualization
* Drawing data-driven conclusions from educational datasets
* Writing clean, reproducible, and well-documented R code

This project was completed as part of a Data Science assignment to strengthen practical skills in statistical analysis, visualization, and reproducible research using R.
